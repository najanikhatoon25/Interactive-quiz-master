# 🎯 Interactive Quiz

An interactive quiz application built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, receive their score at the end, and restart the quiz.

## 🚀 Features

- Start quiz screen
- Multiple-choice questions
- Score calculation
- Next question navigation
- Restart quiz functionality
- Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
